# ZJU_ISEE_Lectures
## 前言
这个仓库用来储存一些我自己做的回忆卷、我的课程笔记与实验报告。  
这个仓库建立的初衷是给后来的学弟学妹们提供一点可以用来平时和复习时参考的资料，尽管可能并不完善。  
我是一个很懒狗的人，所以建立仓库的时间一直推迟。但是大二下学期结束之后，我终于开始准备建立它。它建立的直接原因是，在信电学院电子电路设计实验II这门课上，由于FTP客户端的Altium Designer 9软件包含有病毒，我不得已重装了电脑的系统，丢失了所有的文件，所以我决定在这一学期结束之后立即建立这个仓库，至少能给后来人留下一点点东西。
***
## 包含内容
包含大一部分基础数理课程的笔记和实验报告，大二部分课程的实验报告、A4开卷材料与历年题。  
持续更新中~
